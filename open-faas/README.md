## ABSTRACT
<a href="https://www.openfaas.com/">OpenFaas</a> is an open source serverless framework
<br/>
that enables developer to run functions on any cloud without fear of lock-in.
<br/>
It is written in the <a href="https://go.dev/">Go</a> programming language.

<br/>
OpenFaaS makes it easy for developers to deploy event-driven
<br/>
functions and microservices to Kubernetes.

## Prerequisites
* A kubernetes Cluster (you can use kind or minikube for local development)
* kubectl utility
* OpenFaas CLI

## Usage
Clone this repo, go to the specific sub-folder and follow the readme!
